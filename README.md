# 🐾 Animal Rescue Game 🐾

A fun, interactive web-based game where players rescue various animals to earn points! Help save cute animals before they disappear!

## Game Features

- **60-second rescue missions** - Race against time to save as many animals as possible
- **7 different animal types** with varying point values:
  - 🐶 Puppy (10 points)
  - 🐱 Kitten (10 points)
  - 🐰 Bunny (15 points)
  - 🐹 Hamster (20 points)
  - 🦔 Hedgehog (25 points)
  - 🐸 Frog (30 points)
  - 🐧 Penguin (50 points)
- **Dynamic spawning system** - Animals appear randomly with different rarity levels
- **Responsive design** - Works on desktop and mobile devices
- **Engaging animations** - Smooth hover effects and rescue animations
- **Scoring system** - Track your progress and try to beat your high score

## How to Play

1. **Start the Game**: Click "Start Rescue Mission" to begin
2. **Rescue Animals**: Click on any animal card to rescue it and earn points
3. **Beat the Clock**: You have 60 seconds to rescue as many animals as possible
4. **Earn Points**: Different animals are worth different points (rarer animals = more points)
5. **Game Controls**:
   - **Pause**: Pause the game at any time
   - **Reset**: Reset the game to start over
   - **Space Bar**: Start the game (keyboard shortcut)
   - **Escape**: Pause the game (keyboard shortcut)

## Getting Started

### Option 1: Open Directly
Simply open `index.html` in any modern web browser.

### Option 2: Local Server (Recommended)
```bash
# Navigate to the project directory
cd front_animals_rescuers

# Start a local server (Python 3)
python3 -m http.server 8000

# Or with Python 2
python -m SimpleHTTPServer 8000

# Or with Node.js (if you have http-server installed)
npx http-server -p 8000
```

Then open your browser and go to `http://localhost:8000`
